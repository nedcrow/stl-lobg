팀프로젝트 : League of Battle Ground (LOBG)
팀명 : Countdown
팀원 : 황혁(팀장), 오정원, 황하림
제작기간 : 20/09/23 ~ 20/11/02 (39일)
영상제작 : 황하림

엔진 버젼 : Unreal Engine 4.25.4



조작법

1. 타이틀 레벨
이름을 입력하지 않으면 임의로 정해진다.
서버 생성 버튼 : 리슨 서버 실행하여 로비로 이동.
게임 참가 버튼 : 써놓은 ip 주소로 접속 시도.

2. 로비 레벨
자신은 녹색 사각형으로 표시. 자신의 패널에 마우스를 올리고 우클릭하면 다른 팀으로 이동할 수 있다.
채팅 입력 가능.
호스트는 게임 시작 버튼으로 배틀 맵으로 이동 가능.

3. 배틀 레벨
레벨 이동과 동시에 캐릭터 선택.

wasd : 이동
qe : 몸 기울이기
마우스 이동 : 시야 회전
마우스 왼쪽 클릭 : 사격
마우스 오른쪽 클릭 : 조준
스페이스 : 점프
Tab : 스코어 표시

우물 주변에서 F : 상점 열기. 클릭하여 탭 전환 및 상품 구매.

대형 타워가 파괴되면 결과 표시. 이후 로비 레벨로 이동.
